# Deploy The Full Weasel (Netlify Drop)

1. Build the app:
   - `npm run build`
2. Open [Netlify Drop](https://app.netlify.com/drop).
3. Drag the generated build folder onto the page:
   - Vite projects: `/Users/andrew/Projects/The_Full_Weasel/dist`
   - Create React App projects: `build` (not used in this repo)
4. Wait for upload to finish and copy the live URL to share.

## Android install (Chrome)

1. Open the shared URL in Chrome on Android.
2. Tap the browser menu (three dots).
3. Tap **Add to Home screen**.
4. Confirm install; launch from the new home icon.

## Notes

- Audio starts after the first tap (mobile autoplay policy).
- MP4 backgrounds are included and may be heavier on cellular connections.
